# MClock

Configuration via feature toggles
- Kaizen time colours
- Invert Green/red colours
- Disable colours on weekends
- Work Start / end times
- Kaizen time start
- Lunch start / end times

Discord Rich Presence
- Enable or disable
- Disable on weekends
- Updates depending on time of day e.g. lunchtime or kaizen time
- Can be force disabled if enabled in the settings, but won't show if it's disabled

Notifications
- enable or disable via settings
- Lunch time start / end notification
- end of day patch reminder
